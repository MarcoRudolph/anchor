| Boundary | Description |
|----------|-------------|
| Webapp (user JWT) → Edge Function | RLS-scoped, user-tier verbs only |
| Hermes (service-role + x-hermes-secret) → Edge Function | RLS bypassed; explicit handler authz |
| Browser → magic-link callback | Untrusted code exchange → server session mint |
| Edge fn → Google refresh token at rest | Encrypted material crosses storage boundary |
